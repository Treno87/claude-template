# Voice Notification Skill

음성으로 사용자에게 알림을 제공하는 스킬입니다. 작업 완료, 승인 대기, 에러 발생 등의 상황에서 음성으로 알려줍니다.

## 사용 시점

1. **TDD 사이클 완료 후**: 테스트 통과 시 "테스트 통과, 승인 대기 중"
2. **사용자 입력 필요**: "사용자 입력이 필요합니다"
3. **긴 작업 완료**: "작업이 완료되었습니다"
4. **에러 발생**: "에러가 발생했습니다"
5. **"go" 대기 중**: "다음 단계 준비 완료, go를 말씀해주세요"

## 도구

이 스킬은 Windows의 내장 음성 합성(SAPI) 또는 Python의 pyttsx3 라이브러리를 사용합니다.

## 예시

```python
# notify_voice.py 스크립트 사용
python ~/.claude/skills/voice-notification/notify_voice.py "작업 완료"
```

## 설정

- **음성 속도**: 기본 150 (조절 가능)
- **볼륨**: 100% (최대)
- **언어**: 한국어 우선, 없으면 영어
